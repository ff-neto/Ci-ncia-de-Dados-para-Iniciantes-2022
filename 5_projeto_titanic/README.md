## Sobre o Projeto

1. Definir a coluna: nome, idade, sexo, classe e sobreviveu
2. Filtrar nome por ordem alfabética
3. Filtrar sexo somente feminino, classe (somente 1º classe) e se sobreviveu
4. Salvar o arquivo em csv.
